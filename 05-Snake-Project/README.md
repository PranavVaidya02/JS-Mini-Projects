# Snake Game

A classic Snake Game built using **HTML, CSS, and JavaScript** to practice JavaScript logic, DOM manipulation, timers, keyboard events, and game development fundamentals.

## Features

* Control the snake using the arrow keys
* Random food generation
* Snake grows after eating food
* Live score tracking
* High score saved using Local Storage
* Real-time game timer
* Game Over screen
* Restart the game without refreshing the page
* Responsive game interface

## Concepts Practiced

* DOM Manipulation
* Event Handling
* Keyboard Events
* CSS Grid
* Arrays and Objects
* Game Loop using `setInterval()`
* Collision Detection
* Local Storage
* State Management
* Conditional Statements
* Functions

## What I Learned

While building this project, I learned how to create a simple game using JavaScript by managing the game state, rendering the snake on every frame, handling keyboard input, detecting collisions, generating random food positions, tracking scores, using timers, and storing the high score with Local Storage. This project also helped me understand how game loops and continuous rendering work in browser-based games.
