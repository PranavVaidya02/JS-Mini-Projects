# Kanban Board

A simple Kanban Board built using HTML, CSS, and JavaScript to manage tasks across different stages while practicing DOM manipulation, drag-and-drop functionality, localStorage, and dynamic UI updates.

## Features

- Create new tasks using a modal
- Drag and drop tasks between **To Do**, **In Progress**, and **Done** columns
- Delete tasks
- Automatically update task counts for each column
- Save tasks using localStorage
- Restore tasks after refreshing the page
- Responsive and clean dark-themed interface

## Concepts Practiced

- DOM Manipulation
- Event Handling
- Drag and Drop API

- Functions
- Arrays & Objects
