# Calculator

A simple calculator built using HTML, CSS, and JavaScript to perform basic arithmetic operations while practicing JavaScript logic, DOM manipulation, and state management.

## Features

- Perform Addition, Subtraction, Multiplication, Division, and Modulus operations
- Supports decimal numbers
- Clear all calculations (AC)
- Delete the last entered digit (DEL)
- Displays the current expression and result separately
- Responsive user interface

## Concepts Practiced

- DOM Manipulation
- Event Handling
- State Management
- Functions
- Conditional Statements
- String Manipulation
- JavaScript Operators
- Responsive Layout
