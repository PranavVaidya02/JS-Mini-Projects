# Tic Tac Toe Game

A browser-based Tic Tac Toe game built using HTML, CSS, and JavaScript.

Two players take turns placing their marks on a 3×3 game board. The game checks for winning combinations and determines whether a player has won or the match ends in a draw.

## Features

- Interactive 3×3 Tic Tac Toe board
- Two-player gameplay
- Alternating player turns
- Automatic winner detection
- Draw detection
- Restart/reset functionality
- Game status messages

## How to Play

1. Open the game in a browser.
2. Player X starts the game.
3. Players take turns selecting empty cells.
4. The first player to place three marks in a row, column, or diagonal wins.
5. If all cells are filled and no player wins, the game ends in a draw.
6. Use the restart/reset button to play again.
